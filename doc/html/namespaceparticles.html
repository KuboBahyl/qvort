Detailed Description

The particles aspect of the code. At present this evolves either fluid or inertial particles in the code. The type of particle is set by the particle_type parameter in run.in Fluid/Inertial particles use the normal fluid velocity. The initial setup of the particles is set throught the initp parameter.
