Detailed Description

mesh structure - normal and superfluid velocities are calculated on the mesh which runs from -box_size /2 to box_size/2 to with mesh_size^3 points, note by setting mesh_size>0 mesh is switched on mesh velocities calculated every mesh_shots timesteps
